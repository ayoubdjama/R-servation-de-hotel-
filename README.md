🏨 Système de Réservation d’Hôtel – Royal Plaza

📌 Description

Ce projet est une application web de réservation d’hôtel développée dans le cadre d’un projet universitaire à l’Université de Djibouti.

L’application permet aux clients de consulter les chambres disponibles et de faire des réservations en ligne, tandis que les administrateurs peuvent gérer les chambres, les clients et les réservations via une interface dédiée.

---

🎯 Objectifs

- Automatiser la gestion des réservations
- Éviter les doubles réservations
- Centraliser les données (clients, chambres, réservations)
- Offrir une interface simple et intuitive

🚀 Fonctionnalités

👤 Côté client

- Consulter les chambres disponibles
- Rechercher selon les dates
- Réserver une chambre
- Recevoir une confirmation

🔐 Côté administrateur

- Se connecter (authentification)
- Gérer les chambres (CRUD)
- Gérer les réservations
- Gérer les clients
- Voir les statistiques
